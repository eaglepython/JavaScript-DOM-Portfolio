1. **Clone the repository**
   ```bash

   cd JavaScript-DOM-Portfolio
   ```

   ```bash

   # Python
   python -m http.server 8000
   
   # Node.js
   npx live-server

   php -S localhost:8000
   ```

   ```
   http://localhost:8000
   ```
